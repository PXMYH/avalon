# cyclops

Slackbot for Eagle Eye project integration

## Simplest Way to Get Started

```bash
npm install
token=<bot_token> npm start

# expose local server
lt --port 8000

# or use ngork
```

## Botkit Starter Kit

```bash
npm install
# create slack app and fill in .env file with client ID and secret
node .
# go to https:/localhost:<port>/
```
